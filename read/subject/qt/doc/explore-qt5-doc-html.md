可以了解到，關於「[qt5-doc-html](https://packages.ubuntu.com/bionic/qt5-doc-html)」這個套件，並沒有包含到所有的Qt文件頁面，

例如在「[如何安裝 Qt 相關文件 (HTML版)](install-qt5-doc-html)」

提到的「/usr/share/qt5/doc/qtwidgets/qtwidgets-index.html」和「/usr/share/qt5/doc/qtwidgets/widgets-tutorial.html」

就沒有包含在「[qt5-doc-html](https://packages.ubuntu.com/bionic/qt5-doc-html)」這個套件。

執行下面指令

``` sh
$ dpkg -L qt5-doc-html | grep qtwidgets
```

沒有任何顯示。

執行下面指令

``` sh
$ dpkg -S /usr/share/qt5/doc/qtwidgets/qtwidgets-index.html
```

顯示

```
qtbase5-doc-html: /usr/share/qt5/doc/qtwidgets/qtwidgets-index.html
```

執行

``` sh
$ dpkg -S /usr/share/qt5/doc/qtwidgets/widgets-tutorial.html
```

顯示

```
qtbase5-doc-html: /usr/share/qt5/doc/qtwidgets/widgets-tutorial.html
```

表示這兩個頁面都來自另一個套件「[qtbase5-doc-html](https://packages.ubuntu.com/bionic/qtbase5-doc-html)」

那為啥在「[如何安裝 Qt 相關文件](install-qt5-doc-all)」，提到

安裝了「[qt5-doc-html](https://packages.ubuntu.com/bionic/qt5-doc-html)」這個套件，

就會安裝大部分的Qt文件。

可以執行下面指令，就能了解到，

``` sh
$ apt-cache show qt5-doc-html | grep '^Depends:'
```
